## Built with

* [ExpressJs](hhttps://expressjs.com/) - The web framework used
* [MongoDB](https://www.mongodb.com/1) - Database
* [Mongoose](https://mongoosejs.com/) - Elegant mongodb object modeling for node.js

## Getting Started
```
1. clone this repository
2. npm install
3. setup the .env
5. run start.dev
```
